Дипломный проект Сергея Толокунского

I developed this application as a 'graduation project' for the Java Pro course provided by Prog.kiev.ua. The application loads the spreadsheets from Google Drive and fetches data about students, following the pre-defined format. Students can enter their mobile phone number to get the video for the courses they admitted.

I used these technologies and standards:

  • Google Authorization API
  • Google Sheets API
  • Java SE
  • Java EE
  • Spring MVC Framework
  • JSP
  • Java Mail
